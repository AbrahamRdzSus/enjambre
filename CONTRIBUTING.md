# Contributing

Gracias por contribuir a ENJAMBRE.

## Reglas

- No subas API keys ni secretos.
- No incluyas código copiado de proyectos incompatibles.
- No agregues dependencias con licencias problemáticas sin discusión.
- No implementes funciones de distillation/training con outputs de proveedores propietarios.
- Todo cambio que ejecute comandos debe tener confirmación humana o sandbox.

## Developer Certificate of Origin

Al contribuir confirmas que tienes derecho a enviar tu contribución bajo la licencia del proyecto.
